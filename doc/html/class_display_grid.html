◆ highlightSelection()

DisplayGrid::highlightSelection ( const int  iMin,
const int  jMin,
const int  iMax,
const int  jMax 
)

Only highlights from row iMin to row iMax, and from column jMin to column jMax. This function assumes iMin <= iMax and jMin <= jMax, and does not check the values.

This function first calls DisplayGrid::clearHighlight() to ensure that there is only one highlighted selection at once. Then it sets isAnythingHighlighted to true, before calling Refresh() on the DisplayGrid object.

Parameters
iMinThe zero-indexed upper row of the selection visually (lower numerically)
iMaxThe zero-indexed lower row of the selection visually (higher numerically)
jMinThe zero-indexed leftmost column of the selection
jMaxThe zero-indexed rightmost column of the selection

◆ clearHighlight()

DisplayGrid::clearHighlight ( bool  refresh = true)

This function only does anything if isAnythingHighlighted . This is to save on computation if there is nothing already highlighted. If there is anything highlighted at all, this function loops through the entire grid to unhighlights all Tile objects, even if they are not highlighted. Then isAnythingHighlighted is set to false.

Parameters
refreshThis parameter has a default value of true, and tells the function whether or not to call Refresh() on the DisplayGrid object
